- 환경
- 디지털 중독 관련 기술
- 청각장애인 대중교통 위치 파악: 버스 같은 경우 위치까지 추적해서 버스 안내 음성을 자막으로 보여줌 지하철은 음성 신호를 자막으로 변환, 하차 지점 도움 (우진)
- 그 외 전반적인 사회적 이슈 목록 검토

구현 아이디어가 구체화 된 감정 위험도 분석 or 대중교통 보조

**대중교통 하차 보조**
- 서비스가 비장애인분들에게도 이점을 가지고 있다면 이점을 가지지 않을까
- 위치 기반 시스템 -> 장애인 접근성을 좋게 한다
- 탑승 도중에, 하차할 역 같은 부분을 알려준다
- 청각장애인 분들은 파악이 어려울 것 같다
- 대중교통이 지역마다 다르다. 지하철이라는 대중교통에 익숙하지 않은 사람들 (외국인이나 지하철 탑승 시 혼잡한 상황에서 통로에 모니터가 없고, 화면으로만 현 위치가 어딘지 안내를 함, 접근성이 주어진 환경마다 다른다)
- 장애인 접근성, 고령자나 디지털 소외
- 음성 인식 -> 자막으로 딴다
- 지하철을 목표로 하면 지하에서 GPS가 잘 잡히지 않는 문제? -> 저희가 구축할 필요 없이 지도나 철도 정보 api 가져다 쓰면 될 것 같음
- 프로젝트 목표에 맞는 구체적인 기술 및 프로젝트 규모 결정에 난항

**감정 위험도 분석**
- 개인의 일기, 상태 기록 -> 자연어 처리 기술 활용 -> 감정 상태나 심리상태
- 감정 데이터에 맞추어 파인튜닝
- 감정 상태 추출 -> 간단한 부가 서비스로 연결
- 감정 일기
